Hi!, this is my website Boilerplate

by Horacio Herrera - media designer - hherrerag.com - @hhg2288

version 0.1

This boilerplate uses:

  • CSS Preprocessed with SASS
  • Compass
  • ZURB Foundation Framework 4
  • Foundation Interchange image Library (js)
  • Modernizr
  • Normalize
  • sass-mq()
  • Bourbon SASS Mixins
  • Jquery (latest)
  • Jquery LazyLoad Plugin

all dependencies are managed by Bower.

This is the base file structure that I use when I start any website project. Hope you like it!.
